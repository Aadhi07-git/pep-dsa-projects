# Cricket Tweet Sentiment Analysis

## Project Overview
This project performs sentiment analysis on cricket-related tweets using Python.
The goal is to classify tweets into positive, negative, or neutral sentiments and
visualize the distribution.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Steps Performed
1. Data loading and cleaning
2. Sentiment labeling
3. Exploratory Data Analysis (EDA)
4. Visualization using bar chart and box plot

## Output
- Sentiment distribution bar chart
- Box plot of sentiment scores
